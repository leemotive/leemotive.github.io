# 根目录

拿eslint的仓库为例

- 在工作区目录下的某个子目录，想知道版本库目录路径
```bash
$ git rev-parse --git-dir
/path/to/eslint/.git
```

- 显示工作区根目录
```bash
$ git rev-parse --show-toplevel
/path/to/eslint
```

- 相对于工作区根目录相对路径
```bash
$ git rev-parse --show-prefix
lib/rules/
```

- 当前目录相对工作区目录回退路径
```bash
$ git rev-parse --show-cdup
../../
```
